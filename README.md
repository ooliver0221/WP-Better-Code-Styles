# WP-Better-Code-Styles
🚀 **一个为 WordPress 打造的轻量级样式优化插件，专为提升代码阅读体验而生。**
本插件旨在解决使用 **Code Block Pro** 插件时的样式繁琐配置问题，并为正文中的 **内联代码** 提供美观的洛谷风格化修饰。

## ✨ 核心特性

- **🚀 全自动化注入**：无需在每个代码块的“高级 - 额外的 CSS 类”中手动输入类名。插件通过模糊匹配自动识别并应用样式。
- **🎨 洛谷风格内联代码**：正文中的 `code` 标签自动获得加大背景、圆角及精致的边框，视觉上更加突出且易读。
- **📐 完美圆角支持**：强制修正 Code Block Pro 的圆角切割问题，确保标题栏 (Header) 和底部栏 (Footer) 完美契合容器圆角。
- **⚡ 极高优先级**：采用 `html body` 权重增强及 `9999` 钩子优先级，确保能强力覆盖绝大多数 WordPress 主题的默认样式。
- **🛡️ 防样式污染**：智能排除代码块内部的 `code` 标签，确保代码块内的高亮颜色不受内联代码样式的干扰。

---

## ⚙️ 自定义设置

如果你想修改颜色或圆角大小，可以直接在设置中找到 WP-Better-Code-Styles 代码样式设置 并修改。
